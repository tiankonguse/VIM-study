# VIM STUDY

这个项目目的在于学习 vim.

虽然网上已经有很多关于vim的教程了，可是自己学习时还是要记笔记的。
这对于自己来说，收获绝对比只看别人的教程大。



## 声明：

下面的操作全部在命令行中输入。

```
如果要输入ab，我会写为   ->ab

如果要先操作c,再操作d,我会写为  ->c  ->d

如果需要先操作e,然后回车，我会写为 ->e /
```
